# tnk_flutter_pub

티엔케이팩토리 지면광고 flutter plugin 입니다.

flutter plugin에서는 아래의 기능을 사용 하실 수 있습니다.

* 전면 광고(Interstitial Ad)
* 네이티브 광고(Native Ad)
* 동영상 리워드 광고(Video Reward Ad)
